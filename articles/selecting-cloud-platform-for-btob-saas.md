## 選定基準

認証も悩んでるけど、諸々を Firebase でいくか AWS でいくかをまだ悩んでる。
なんとなく AWS に傾いてきてるけど、構成がいまいち決まりきらないから費用の不安が拭えない。

- 方針 1.最初 Firebase でやって、スケールしたら AWS に切り替える

  - スケールしたときに切り替えるのが地獄そう

- 方針 2.最初から AWS でいく
  - 費用を抑える方法を探したとしても、個人資金の範囲で収まるかどうか
